# Wing PayWay - KHQR Payment Gateway

Simple and clean payment gateway for Wing Bank KHQR payments.

## Features

- 💰 Quick amount selection ($1, $2, $5, $10, $20, $50)
- 📱 Real-time QR code generation
- ⏱️ Automatic payment detection
- 🎯 Clean and professional UI
- ✅ No backend required - pure HTML/JavaScript

## Usage

Just open `payway.html` in your browser. No installation needed!

## How it works

1. Enter payment amount or click quick select
2. Click "Generate Payment QR"
3. Customer scans QR with Wing Mobile or any KHQR app
4. Payment confirmation appears automatically

## Live Demo

Open `payway.html` in any modern browser.

## API

This project uses the KHQR Payment API for QR generation and payment verification.
